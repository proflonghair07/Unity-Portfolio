About Me

My name is Murphy Vaughn, and I am a Nashville-based developer with three years of experience creating games in the Unity game engine. I currently have ten released games and countless other prototypes. My expertise primarily lies in 2D game development, although I also have experience in 3D game design. I possess a solid grasp of C# and a knack for quickly adapting to new tools, which allows me to prototype rapidly.

My journey in the world of programming began with web design, where I learned the fundamentals through JavaScript. This initial foray into programming eventually led me to explore C# and the Unity game engine. What started as a small side hobby quickly evolved into a full-blown passion as I discovered how satisfying and rewarding the marriage of various creative and technical aspects of game design can be.

My main passions are scripting, level design, and sound design, but there are few areas in the field that don't intrigue me. I'm excited to continue growing, learning, and creating in this constantly evolving field. If you're looking for a dedicated and passionate game developer, let's connect and create something amazing together.

Candidus Caves

Candidus Caves is a 1-bit esque Metroidvania game that features three unlockable upgrade abilities. The player plays as a young boy named Magnus who is taking his first trip to the caves with his father after coming of age. Upon arrival the lift that allows access to the caves breaks down. Magnus is tasked with collecting enough "Reparare Rhodonite" so that his father can restore it to working order.

Various different tools where utilized in the making of this game. With a few exceptions the majority of this game's art was sourced from asset packs on itch.io. To achieve a more cohesive look all of the various assets were converted to 1-bit using this dithering tool created by TimHeiGames. Sprites were then recolored in Aseprite as needed. The splash screen logo animation and other various effects were created with CodeManu's pixel art tools.

A downloadable version of this project can be found here. The project's GitHub repository can be found here.

Bubble Paths

Bubble Paths is a relaxing underwater platformer with light story and light puzzle elements. The player plays as Emster who is a seasoned diver and against his better judgment and with a bit of desperation is going on another dive to secure much needed funds.

The main mechanic in this game is a somewhat unique limited breath one in which the player has to collide with bubbles that move along set paths to refill thier oxygen counter. The game features 8 compact single screen puzzle levels.

A browser build of this project can be found here. The project's GitHub repository can be found here.

Kit Twins III

Kit Twins III was my submission to the 2023 GMTK Game Jam. As the title might suggest, it is my third entry to this jam with Kit Twins I being my first completed game. I decided to do a sequel to the first game on the second year because I thought it would be nice to have a fairly direct comparison between my first and most recent game to see how far I had grown. I have enjoyed this process and plan on doing it for the forseeable future.

The theme for this jam was "Roles Reversed". The main mechanic in all Kit Twin titles is mirrored movement between the player and the active Kit Twin. The goal is to get the twin to a campfire to progress to the next area. For the roles reversed theme I had the platformer container mirror the movement of the player instead of the twin with the player manuevering it around the twin and get it to the campfire. Each twin's platform container was made utilizing Unity's tilemap system and I was pretty pleased with the results.

A browser build of this project can be found here.
